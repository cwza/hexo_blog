## 安裝
homebrew會裝到/usr/local/
```
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```
## 指令
``` bash
 #homebrew檢查
brew doctor
 #homebrew更新
brew update
 #列出安裝的module
brew list
```
``` bash
 #module會裝到/usr/local/opt/xxx
 #安裝xxx模組
brew install xxx
brew uninstall xxx
 #更新xxx模組
brew upgrade xxx
```
