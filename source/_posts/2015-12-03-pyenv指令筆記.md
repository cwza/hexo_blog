## MAC底下安裝
記得先裝xcode command line tools
pyenv會裝到/usr/local/opt/pyenv
``` bash
brew install pyenv
```
修改bash_profile加上
``` bash ~/.bash_profile
export PYENV_ROOT=/usr/local/opt/pyenv
eval "$(pyenv init -)"
```

## 指令
``` bash
#列出所有可以安裝的python版本
pyenv install --list

#安裝python 3.4.2
pyenv install 3.4.2

#反安裝python 3.4.2
pyenv uninstall 3.4.2

#列出目前安裝的python版本
pyenv versions

#set目前folder下的python版本to 3.4.2 會在folder底下生出.python-version檔案
pyenv local 3.4.2

#set目前folder下的python版本to system
pyenv local system

#顯示目前folder下的python版本
pyenv local

#unset目前folder下的python版本
pyenv local --unset

#set所有shell下的python版本to 3.4.2
pyenv global 3.4.2

#顯示目前folder下的python版本
pyenv global

#set目前folder下的python版本to system
pyenv global system
```
