``` bash
#移除沒用到的import
Ctrl + Shift + o

#區域向左縮排
Shift + Tab

#區域向右縮排
Tab

#全project下檔案名稱搜尋
Ctrl + Shift + r

#該文件下function搜尋
Ctrl + o

#全project下字串搜尋
Ctrl + Shift + L

#搜尋開啟文件
Ctrl + e

#搜尋該function所有被呼叫到的地方
Ctrl + Shift + g

#將編輯視窗最大化
Ctrl + m

#將游標移到上一個地點
Alt + left or right

#移動整行
Alt + up or down

#加註解
Ctrl + /

#refactor變數or function name
Alt +　Shift + r

#開啟宣告
F3 or Ctrl + mouse left click
```
